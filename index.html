BLUEbikes trip data in May 2019

Bluebikes, is a bicycle sharing system in the Boston, Massachusetts metropolitan area.

Since the launch of the service in 2011 rich system data has been made available to the public free of charge.

This web map visualizes the trip dataset from May 2019. It shows all used start stations of BLUEbikes and draws flow lines to all destination stations.

The color and width of the lines of the flow map indicate the number of times a specific route has been used.

BLUEbikes data source: https://www.bluebikes.com/system-data
